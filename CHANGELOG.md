# Changelog

All notable changes to this project will be documented in this file.

This project adheres to semantic versioning.

---

## [0.1.0] - 2026-03-03

### Added
- Initial release of `PinchTranslatorService`
- Real-time speech-to-speech translation integration with Pipecat
- Daily transport foundational example
- Editable install support (`pip install -e .`)
- PyPI distribution as `pipecat-plugins-pinch`

### Notes
- Tested with `pipecat-ai >= 0.0.50`
- Python >= 3.10 supported (tested with 3.13.2)
